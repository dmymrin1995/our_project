Для синхронизации с гит надо
1	создать папку где будет проект	чтобы знать где будет проект
2	открыть консоль	чтобы вводить команды
3	git init	создание гит структуры
4	git remote add origin https://github.com/dmymrin1995/our_project.git	создание связи с проектом
5	git pull origin ветка	скачивание нужной ветки с ресурса
6	git checkout ветка	переключение на нужную ветку

Для запуска всего этого надо
1. запустить сервер в консоли node project/server.js
2. перейти по ссылке в консоли

для первого автосоздания html на основе pug
1. установить пуг npm install pug
2. установить пуг для командной строки npm install pug-cli -g
3. если pug ругаеются на безопасности запустить Set-ExecutionPolicy RemoteSigned

для автосоздания html на основе pug
1. перейти в папку проекта cd project
2. запустить команду pug ./ -P -w
3. если отменить консоль надо вводить команду по новой
